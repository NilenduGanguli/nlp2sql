Produce your output as a JSON array of objects with exactly these fields:

- value_set_guide: What a value set means, when each value appears, how to filter by it
- join_strategy: How to traverse from one table to another, which keys to use, what the path means
- query_pattern: A reusable SQL query pattern for a common business question
